DrillTek

A Borehole Management Application for Diamond Drilling in the Mining Industry

Joshua Smiles

Abstract

DrillTek is an open-source, web-based solution for managing and logging boreholes in extractive industry. It is designed to be used inside mining operations and exploration companies. The application is three-tier, comprising a Svelte-based client, an API service built using Django Rest Framework, and a PostgreSQL database. Users can create borehole programs comprised of individual boreholes and upload geological logging information associated with them. This information is then available for access via 3D orebody modeling software.
