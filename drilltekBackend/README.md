# Drilltek-Backend

## Python Django Installation (Windows)

To install Django ahead of running this server please ensure python is installed on the OS using

```
python --version
```

If Python is not installed, please install from the microsoft store or from [Python.org](https://www.python.org/downloads/)

Pip is required to install Django. Ahead of installing Django, please check pip version using 

```
pip --version
```

Once python and pip are installed, install Django using 

```
python -m pip install Django
```

And check Django version using 

```
python -m django --version
```
